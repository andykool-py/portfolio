# Portfolio

A clean, responsive personal portfolio website that can be hosted directly on GitHub Pages.

## What to edit

- Replace `Your Name` and `YN` in `index.html`
- Update the about text, project names, and contact links
- Swap the placeholder email and social URLs with your real links
- Tweak colors, spacing, and copy in `styles.css`

## Run locally

Open `index.html` in your browser, or use VS Code's Live Server extension for a smoother preview.

## Host on GitHub Pages

1. Create a new repository on GitHub, for example `portfolio`.
2. Put these files in the root of the repo.
3. Commit and push to your main branch.
4. In GitHub, open the repository settings.
5. Go to `Settings > Pages`.
6. Under `Build and deployment`, choose `Deploy from a branch`.
7. Select the `main` branch and the `/ (root)` folder.
8. Save. GitHub will publish the site after a short build delay.
9. Your site will appear at `https://<your-github-username>.github.io/portfolio/`.

## Optional custom domain

If you own a domain, add a `CNAME` file with your domain name and configure the DNS records in your domain provider.
